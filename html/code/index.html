DOCUMENTACIÓN Y AYUDA

Este es un programa de análisis musical en Python. El programa utiliza la biblioteca customtkinter para crear una interfaz gráfica de usuario (GUI) para el usuario. El programa permite al usuario cargar un archivo de audio, analizarlo y mostrar los resultados en la interfaz gráfica de usuario.

El programa tiene las siguientes funciones:
- add_analisis(): agrega el análisis del archivo de audio seleccionado
- save_data(): guarda los datos analizados en un archivo de texto
- help_web(): muestra una página web de ayuda
- add_nota(): agrega una nota


El programa tiene una clase App, que es la clase principal de la aplicación. El método constructor de la clase App se llama automáticamente cuando se crea un nuevo objeto de la clase.

El programa utiliza los siguientes módulos:
- os: proporciona funciones para interactuar con el sistema operativo.
- warnings: proporciona funciones para mostrar advertencias.
- webbrowser: proporciona funciones para abrir páginas web en el navegador predeterminado del usuario.
- tkinter.filedialog: proporciona funciones para mostrar cuadros de diálogo para abrir y guardar archivos.
- customtkinter: una biblioteca personalizada que proporciona clases para crear una GUI.


Para ejecutar el programa, simplemente debe ejecutar el archivo "Main.py". A continuación, se muestra una descripción detallada de cómo usar el programa:

## 1. Carga de un archivo de audio


Para cargar un archivo de audio, haga clic en el botón "Analizar". Se abrirá un cuadro de diálogo para que seleccione el archivo de audio que desea analizar. Seleccione el archivo de audio y haga clic en el botón "Abrir". El programa analizará el archivo de audio y mostrará los resultados en la ventana principal.

## 2. Guardar los datos analizados


Para guardar los datos analizados en un archivo de texto, haga clic en el botón "Guardar". Se abrirá un cuadro de diálogo para que seleccione la ubicación y el nombre del archivo de texto. Seleccione la ubicación y el nombre del archivo de texto y haga clic en el botón "Guardar". Los datos analizados se guardarán en el archivo de texto especificado.

## 3. Mostrar la página de ayuda


Para mostrar la página de ayuda, haga clic en el botón "Ayuda". Se abrirá la página de ayuda en el navegador predeterminado del usuario.

## 4. Añadir una nota


Para añadir una nota, ingrese el texto de la nota en el campo de entrada "Añadir nota" y haga clic en el botón "Añadir nota". La nota se agregará a la lista de notas en la ventana principal.

## 5. Desplazarse por la lista de notas


Si la lista de notas es más larga que el área visible en la ventana principal, puede desplazarse por la lista de notas utilizando el mouse. Haga clic y arrastre el control deslizante en la barra de desplazamiento vertical en el lado derecho de la ventana principal.

## 6. Cerrar la aplicación


Para cerrar la aplicación, haga clic en el botón "X" en la esquina superior derecha de la ventana principal.
